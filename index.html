How To Use

  • Clone the repo, dude!
  • bundle install
  • ruby emojisaic.rb -g [filename.gif] (for a gif) or
  • ruby emojisaic.rb -i [filename.jpg] (for a still image)
  • Wait.
  • Marvel @ your emojisaic!

Options

at least one of these are mandatory:

  • -g [filename] or --gif [filename] - indicate location of the gif to convert
  • -i [filename] or --image [filename] - indicate location of the still image to convert

optional options:

  • -s [int] or --size [int] - height of the emoji in pixels. default is 8. smaller size = longer conversion time
  • -z [int] or --zoom [int] - multiply output file height and width by this number. default is 1. larger = longer conversion time
  • -o [int] or --offset [int] - randomly move emojis by this many pixels. default is 0. lower numbers are generally better, anything above 10 or so is visual gibberish #FreeOffset
  • -c [int] or --coverage [int] - emojis are scored by their average color. however, some emojis have more pixels filled in and will look better with certain kinds of images. without this option set, the program will replace a given pixel area with the closest emoji by color, regardless of how many non-transparent pixels it contains. setting this to 10, for example, will find the 10 closest emojis by color to a given pixel area and pick the one with the highest percentage of pixel coverage. I recommend setting this to between 10 and 30.
  • -q or --quiet - silences all output and updates to the console

utilities:

  • -t or --tmp - deletes all temporary images in the tmp/ folder
  • -m or --map - regenerate emoji color map
  • -h or --help - print these options to the console
